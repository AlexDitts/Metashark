*Проект выполнен согласно ТЗ.
**Перед установкой создать и активировать виртуальное окружение командой
python -m venv venv
**для активации окружения набрать
source venv/bin/activate
**Для установки необходимых зависимостей надо находяс в папке проекта набрать
pip install -r requirements.txt
**Для установки проекта необходимо находясь в папке проекта MetasharkDrf набрать команды
python manage.py makemigrations
python manage.py migrate
**Далее для заполнения базы данных из фикстур набрать
python manage.py loaddata fixtures/total_data.json
**Запускаем проект командой
python manage.py runserver

** Urls для работы в проекте
/swagger/ Документация по проекту с описанием классов, их методов и т. д.
/api/v1/drf-auth/login - логин
/api/v1/drf-auth/logout - логоут
После логина пользователь перенаправляется на страницу с детальным отображением информации о нём. Url -
/accounts/profile/
/api/v1/ - Выводит список ссылок для работы персонала в системе.
/api/v1/status_task/ - Создание в фоновом режиме эксель файла с отчётом по направлениям, дисциплинам, группам и студентам.
/accounts/profile/ - Проверка статуса выполнения задачи формирования отчёта

Для пользователя из группы Curator доступна работа с группами и студентами
Для пользователя из группы Administrator доступна работа с направлениями и дисциплинами, а также
доступна возможность создания в фоновом режиме эксель файла с отчётом по направлениям, дисциплинам, группам и студентам.
Кроме того возможность проверить статус выполнения задачи формирования отчёта
